# transparency-tester

# Requirements ----

- Python 3.11+
  
- pip
  
- Playwright


# Installation ----

- Download https://cdnjs.cloudflare.com/ajax/libs/axe-core/4.10.3/axe.min.js and add it to your source file

pip install playwright

playwright install

pip install streamlit

# Run app ---------

streamlit run app.py
